Особенности работы программы:
- Для удобства заполнения в папке с программой лежит пример заполненного плана - example.docx
- Для запуска программы используйте файл start.bat
- Необходимо быть осторожным при заполнении полей.
Некорректные данные (данные в непредусмотренном формате) могут вызвать "вылетание" программы,
что приведет к потере всех введенных данных.
Рекомендуется проверить корректность работы программы после заполнения полей группы "Общие сведения" (Нажать кнопку "Генерировать" блока "Готово").
- Сгенерированный файл появляется в папке plans каталога программы. Имя файла соответствует имени аспиранта.
- При открытии документа, MS Word может "ругаться" на некорректность документа, но после нажатия "Ок", а затем соглашения на открытие ("Да"), он откроется.
- В программе есть формы 4 таблиц - для случая 4хлетнего обучения. Если срок обучения - 3 года, 4-я таблица не заполняется. В документе она не появится.
